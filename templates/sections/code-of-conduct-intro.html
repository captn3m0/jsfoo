Code of conduct

JSFoo 2013 is a community conference intended for collaboration and networking between developers. We value the presence of every attendee and want everyone to have an enjoyable and fulfilling experience. All attendees are therefore expected to show respect and courtesy to other attendees throughout the workshops and the conference.

All attendees – participants, speakers, exhibitors, organisers and volunteers – at JSFoo 2013 are required to adhere to the following Code of Conduct. Organisers will enforce this code throughout the event.
