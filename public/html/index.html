About

JSFoo is India's only national JavaScript conference series, with editions held in Bangalore, Pune and Chennai. This year's Bangalore edition is a flagship two-day event.

The theme for JSFoo Bangalore 2012 is JavaScript Everywhere. Hear from developers how they and their organizations are pushing the limits of what can be done with JavaScript on the server, in embedded devices, and in its original home in the browser. Talks at JSFoo 2012 will cover:

1. JS in the Browser — Teaching an old dog new tricks

2. JavaScript Everywhere — Using JS in places where JS isn't normally expected

3. JS on Mobile — JavaScript on the go, in the mobile browser and in native apps

4. Server-side JS — JavaScript on the server, a relatively new and increasingly popular option for server-side development

5. Zen of JavaScript — Best practices, war and peace stories, how I did it and how you can too
